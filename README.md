# Tienda de Libros - React

## Descripción
E-commerce de libros hecho con React, Context y SASS. Permite registro, login, añadir al carrito, realizar pedidos y gestionar perfil.

## Tecnologías
- React
- React Router
- Context API
- SASS
- Axios
- Git

## Rutas
- `/` - Home
- `/register` - Registro
- `/login` - Login
- `/profile` - Perfil
- `/cart` - Carrito
- `/product/:id` - Detalle del libro

## Cómo iniciar
```bash
npm install
npm start
